# DamageTiltHelper

> Simple plugin for Paper which sends damage direction to the clients that use
> [DamageTilt mod][mod].

## Requirements

- You need to use [Paper](https://papermc.io/downloads) 1.16.5+.
- Clients require [DamageTilt][mod] mod installed to notice any effect.
